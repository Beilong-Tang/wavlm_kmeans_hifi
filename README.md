## WavLM_Kmeans_Hifi

Generate audio using Kmeans + Conformer (Detokenizer) + Hifi-Gan.

## Pre-requisite

1. Download our conformer ckpt from [here](https://drive.google.com/file/d/1a3LbfVxURgcy7oM3K-IZzQ_Gz6-W_msL/view?usp=sharing) to `./ckpt`
2. Download Kmeans model from [here](https://drive.google.com/file/d/1ckxOx5MVxuHB_6qeEJo1Ae_c-8wZEgY5/view?usp=sharing) to `./ckpt`
3. Download WavLM from [here](https://drive.google.com/file/d/12-cB34qCTvByWT-QtOcZaqwwO21FLSqU/view) to `./ckpt`
